# Mion_pet

一个桌面小宠物，准备把她做成融合怪。运用了[Cindy723/PyRPA: 基于Pyautogui的自动化工具 (github.com)](https://github.com/Cindy723/PyRPA)兄弟的脚本

unity和win混合(因为引用的库版本比较低，不知道为什么就是中文乱码。)

# 使用方法

Releases里下载双击运行Mion_pet.exe即可;

PyRPA的使用方法请看[Cindy723/PyRPA: 基于Pyautogui的自动化工具 (github.com)](https://github.com/Cindy723/PyRPA)的介绍。
